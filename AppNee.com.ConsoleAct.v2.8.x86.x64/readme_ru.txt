
                        	 ConsoleAct Portable

       		                 Системные Требования:  
                          —————————————————————————————————
  Windows XP, Windows Vista, 7, Windows 8, 8.1, 10, Server 2008, 2008 R2, 2012, 
2012 R2, 2016, Office 2010/2013/2016/2019 Объемы выпусков. 
**** Программа НЕ требует какой-либо версии .NET Framework. **** 
 
                        Дополнительные параметры запуска (ключи): 
                          —————————————————————————————————
/win=act 	- Запустить программу в скрытом режиме, активировать Windows и выйти из программы.
/ofs=act 	- Запустить программу в скрытом режиме, активировать Office и выйти из программы.
/wingvlk	- Запустить программу в скрытом режиме, установить ключ Windows и выйти из программы.
/ofsgvlk	- Запустить программу в скрытом режиме, установить ключ Office и выйти из программы.
/taskwin	- Создать в планировщике задачу для переактивации Windows.
/taskofs	- Создать в планировщике задачу для переактивации Office.
/onlinekms	- Использовать внешний KMS-Service.
/disabledefender- Отключить Защитник Windows.
/ru | /en	- Программа будет запущена с русским (английским) интерфейсом.
